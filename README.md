# Educate_Assist

A web application that displays campus-related information, including upcoming events, current weather, and dining hall hours. The project uses HTML, CSS, JavaScript, and Bootstrap, and integrates external APIs for live data.

---

## Features

- Displays upcoming events using the Ticketmaster API
- Shows current weather using the OpenWeather API
- Responsive layout using Bootstrap
- Dynamic content updates using JavaScript
- Simple interaction for displaying dining hall hours

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap 5
- Ticketmaster API
- OpenWeather API

## APIs Used

### Ticketmaster API
Used to fetch upcoming events in Chicago.
